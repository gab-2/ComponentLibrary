# Multi-framework adapters

## Vue

```vue
<script setup lang="ts">
import { SmButton, SmInput, SmCard } from "@sua-marca-ui/vue";
</script>
```

## Angular

```ts
import { SmButtonComponent, SmInputComponent, SmCardComponent } from "@sua-marca-ui/angular";
```

## Svelte

```svelte
<script lang="ts">
import { Button, Input, Card } from "@sua-marca-ui/svelte";
</script>
```

## Web Components

```ts
import "@sua-marca-ui/web-components";
```
